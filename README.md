# Commercial Center

Aplicação web interna de gestão comercial integrada ao Chatwoot.
Desenvolvida com Node.js/Express (Backend) e Vite/React (Frontend).

## Requisitos
- Node.js 18+

## Como rodar o projeto localmente

### 1. Inicializando o Backend
```bash
cd backend
npm install
npm run dev
```

### 2. Inicializando o Frontend
```bash
cd frontend
npm install
npm run dev
```
> O acesso do painel de Módulo 0 ficará disponível em http://localhost:5173/settings/connection
